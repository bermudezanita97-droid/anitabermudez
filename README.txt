Anita founder site - static deploy package

Files:
- index.html
- styles.css
- images/

Fastest deployment path:
1. Create a new project on Vercel.
2. Upload this folder or drag it into a new GitHub repo and connect that repo to Vercel.
3. Once deployed, add anitabermudez.com as a custom domain in Vercel.
4. In Wix domain settings, point the DNS records to the values Vercel gives you.

Notes:
- The contact button uses mailto: anita@anitabermudez.com, so set up forwarding or a mailbox before going live.
- LitFolio screenshots are included under images/.
